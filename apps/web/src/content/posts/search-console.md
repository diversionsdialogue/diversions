**Wat is de kracht van Search Console? Stel: je wilt dat je website beter vindbaar wordt in Google. Zou het dan niet prachtig zijn als Google zelf je wijst op verbeterpunten? En als je niet alleen kunt zien hoeveel bezoekers je website ontvangt, maar ook hoeveel bezoekers je naam hebben gezien in de zoekresultaten? De gratis dienst Search Console van Google biedt dit allemaal.**

Van Google zijn we gewend dat ze ondernemers veel praktische hulpmiddelen bieden. Google Analytics is in gebruik op vrijwel iedere website. Google Adwords helpt veel websites aan een constante stroom bezoekers. Maar vreemd genoeg is [Google Search Console](https://www.google.com/webmasters/#?modal_active=none) (GSC) een stuk minder bekend en veel gebruikt. Hoog tijd dat daar verandering in komt.

## Wat doet Search Console precies?

Search Console geeft je voor een belangrijk deel inzicht in de manier waarop Google je website indexeert en weergeeft in zoekresultaten. Je krijgt informatie over:

- hoe je website door Google wordt gezien
- welke verbeterpunten er zijn
- hoe vaak je site in zoekresultaten wordt vertoond
- welke van je pagina’s het beste scoren
- hoeveel bezoekers je dit oplevert.

Een schat aan informatie voor iedereen die [SEO marketing](/seo-marketing/) serieus neemt.

## De kracht van Search Console benutten

Net als bij Google Analytics moet je je website aanmelden voor de dienst. Dit is in de meeste gevallen heel eenvoudig. Slechts in een beperkt aantal gevallen is hulp van je webbouwer nodig.  Er is een uitgebreid Helpforum beschikbaar. Daar lees je hoe de [activering](https://support.google.com/webmasters/topic/4564315?hl=nl&ref_topic=4581229) van Search Console werkt.

Op deze plek gaan we liever in op de inhoud.

## De zoekanalyse

Dit is het meest tot de verbeelding sprekende onderdeel van Search Console. Standaard toont GSC je hoeveel klikken je per zoekopdracht hebt gekregen. Dat is nuttige informatie, maar ook afleidbaar uit Google Analytics. De echte meerwaarde zit in het aantal vertoningen, de CTR en gemiddelde positie.

## Zoekverkeer

In alle SEO-analyses komt het belang van backlinks en interne links terug. Bij *zoekverkeer* laat Search Console je zien hoe de linkstructuur zowel binnen als buiten je site vorm krijgt. Welke website verwijzen naar je en hoe vaak? En welke pagina binnen je site krijgt de meeste links?

Ook dit levert weer directe input voor je SEO-plan. Je kunt wijzigingen aanbrengen binnen je website om ervoor te zorgen dat je belangrijkste pagina’s de meeste links ontvangen (en niet bijvoorbeeld de disclaimer).

## Zoekopmaak en Crawlen

Search Console heeft ook een technische kant. Zowel bij Zoekopmaak als Crawlen geeft Google je aan wanneer er mogelijkheden zijn om je site beter te indexeren, bijvoorbeeld door fouten op te lossen. Schrik niet als er een aantal foutmeldingen worden getoond: vrijwel iedere site (zeker van enige omvang) krijgt hiermee te maken. Het is vooral belangrijk patronen in foutmeldingen te vinden zodat je er iets aan kunt doen.

Extra van belang bij “crawlen”is het kopje Sitemaps. Zorg er hier altijd voor dat je een werkende, dynamische sitemap deelt. Het helpt Google alle pagina’s van je website te ontdekken.

## Gekoppeld aan GA

In [Google Analytics](https://www.google.nl/intl/nl/analytics/) kun je ervoor kiezen om de gegevens uit Search Console te koppelen. Zo heb je op 1 plek alle gegevens over je vindbaarheid op ee  n rij. Het schema laat je zien hoe dit werkt. Ook als je dit hebt gedaan blijft het verstandig met regelmaat in Search Console zelf te kijken. Je vindt daar nog veel aanvullende informatie die voor je van waarde is.

## Veelgestelde vragen

### Wat is Google Search Console?

Search Console is een dienst van Google voor website-eigenaren. De tool geeft je inzicht in de prestaties en rankings van je website.

### Wat kost Google Search Console?

Niets! Het is een gratis dienst. Je kunt het zelf eenvoudig activeren met behulp van een eenvoudig stappenplan.

### Wat is het verschil tussen Search Console en Analytics?

Google Analytics brengt bezoekersgedrag op je website in beeld. Search console heeft een focus op zoekgedrag binnen Google zelf. Je ziet bijvoorbeeld hoe vaak je site wordt vertoond en hoeveel kliks hieruit voortkomen.
