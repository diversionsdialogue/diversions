## Start - procrastination

Purely because I find caulking to be really frustrating work. At least, sealing with caulk is fine, but actually applying it—I always end up covered in it, and I can never get a straight line, which is really frustrating for me. So I keep putting it off, and when I saw this research, I thought, "Yeah, I still need to buy caulk and finish some sealing." Maybe it’s a bit of a push to finally get that project done.

Exactly, okay, that’s great, especially since you’re such a handy DIYer, but caulking seems to be a challenge for you. What do you find difficult about it?

Well, I always say it’s a skill of its own. For one, it looks simple: you just grab the caulk gun, put the caulk in, draw a straight line, and then smooth it with a wet finger. But, for one thing, I find it quite difficult to apply consistent pressure on the gun, which you need to get a straight line. And then, very often, you either end up with a wavy line, or it doesn’t flow smoothly all the way through. And yeah, I’m quite particular about things. I get a lot of things done well, but a straight caulk line is rarely one of them.

## Experienced DIYer in action

I’m going to caulk the bathroom now. This is the caulk I bought last week. I’m starting in the corner so I can go in one straight line. Then I release the pressure for a moment and set it aside. I also use my finger to smooth it out. There are tools for that, but I prefer using my finger. Then I scrape off the excess. I check to see if I haven’t made the line too thick, or if I need to add a bit somewhere. Here, I’ve gone slightly over the edge. ## Experienced DIYer - disappointing result

"Here’s the final result. I’ve already painted over it. Yeah, I have to say, even on the video, you can still see it. I’m actually not satisfied with the final result. I expected it to turn out a bit better. With wall putty, I’ve never seen this before; I usually get a really smooth result. The caulking part was really easy, though. But I feel like it ended up drying too deep or, well, how should I say, it dried too deeply. So it still looks a bit indented. Caulking itself was very easy and not a problem at all. But yeah, when you look at it now, you can still see some indentations. For now, it’s a temporary solution. I still need to finish it off neatly.
